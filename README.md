# 📚 The Education Tutor for Remote India

This project is a simple AI-based learning tool created to help students, especially in remote areas of India, understand concepts easily.
The idea is to provide explanations in different Indian languages so that students can learn in the language they are most comfortable with.

---

## 🚀 What this project does

- Allows students to ask questions from different subjects  
- Gives easy-to-understand answers  
- Supports multiple Indian languages  
- Works like a basic AI tutor  

---

## 🛠️ Technologies used

- Python  
- Streamlit  
- Requests library  
- OpenRouter API  

---

## ▶️ How to run the project

First install the required libraries:
pip install streamlit requests
Then run the app:
streamlit run app.py

---

## 🌍 Deployment

The application is deployed using Streamlit Community Cloud so it can be accessed online.

---

## 📌 Why this project

Many students in rural areas face difficulty understanding subjects due to language barriers and lack of guidance.

This project tries to solve that by:
- Providing simple explanations  
- Supporting regional languages  
- Making learning accessible anytime
